Extra Information on Icons!

    
To add icons, you'll need to import the google font icons by placing

    
"link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons"" in the head of your HTML file.

        
to add an icon, copy and paste "i class="ICONPLACEHOLDER">menu /i>" into the placeholder text.

        
For example, to add a heart icon, you would use "i class="material-icons">favorite /i>"

        
For a list of icons, visit: https://fonts.google.com/icons.

    






    
Extra Information on Colors!

    
For the colors, I used CSS variables in the :root to make it easier to change the color scheme of the website.

    
For example, to change the color of the navbar, you would change the color of the variable "--color2".

    
I used the website "https://coolors.co/1a1423-372549-774c60-b75d69-eacdc2" to generate the color scheme.
